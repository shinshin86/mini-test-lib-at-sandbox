# mini-test-lib-at-sandbox

A small test library created for learning.

## Run

```sh
yarn testrun
```

## Directory structure of the sample source

Library code.

```
lib
├── mini-test-lib.js
└── test-runner.js
```

Sample source code.

```
src
└── sample.js
```

Test code for the sample source code.

```
test
└── sample.test.js
```
